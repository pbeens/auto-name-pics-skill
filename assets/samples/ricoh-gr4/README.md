# Ricoh GR4 Samples

This is the only sample camera folder currently added.

Place sample photos taken with a Ricoh GR4 in this folder.

Use this directory for camera-specific test images when validating filename generation or metadata handling.

Files in this folder should be renamed in place only.
Do not alter the image data or move the files elsewhere.

Expected filename pattern:

`GR-IV-003414 20260611`

For the current sample files, the embedded number from `R0003565` becomes `003565` and the EXIF date is `20260616`.

Current renamed examples:

- `GR-IV-003565 20260616.JPG`
- `GR-IV-003565 20260616.DNG`
